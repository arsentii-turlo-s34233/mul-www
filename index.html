Nagłówek 1

This is my first responsive website created with HTML and CSS. The page contains a header, navigation menu, main content area, image, footer, and responsive layout for mobile devices.

HTML is responsible for the structure of the page, while CSS is responsible for the appearance. In this exercise I use semantic HTML5 tags such as header, nav, main, article, and footer.

Nagłówek 2

The layout changes on small screens. On desktop, the menu and content are displayed next to each other. On mobile devices, they are displayed one below another, which makes the page easier to read.
